# Любовное послание для Anya Belozerova

Мини-проект без сборщиков (просто HTML/CSS/JS).

## Как запустить
Открой файл `index.html` в браузере (двойной клик) — и нажми кнопку `Открыть письмо`.

## Настройка имени и подписи
В `script.js` в начале есть переменные:
- `friendName` — имя подруги
- `fromName` — как подписаться (например, твое имя/«твой(я) …»)
